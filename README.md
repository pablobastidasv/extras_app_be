##Loading data

### Persons

```Cypher
LOAD CSV WITH HEADERS FROM 
"https://raw.githubusercontent.com/pablobastidasv/extras_app_be/master/data/persons.csv"
AS row
CREATE (p:Person)
SET p = row
```

### Attributes

```Cypher
LOAD CSV WITH HEADERS FROM 
"https://raw.githubusercontent.com/pablobastidasv/extras_app_be/master/data/attributes.csv" 
AS row
CREATE (a:Attribute)
SET a = row
```
